# BantayBaha MVP Roadmap

## Phase 1 - Foundation

- Initialize Spring Boot backend project files.
- Initialize Ionic Angular frontend project files.
- Configure PostgreSQL/PostGIS connection.
- Add JWT authentication and role-based access.

## Phase 2 - Interactive Map

- Add project API endpoints.
- Add Leaflet map screen.
- Display project markers.
- Add search and filters.
- Add project details view.

## Phase 3 - Community Reports

- Add report submission flow.
- Add geotagged photo upload support.
- Add report status tracking.
- Store evidence links through Cloudinary.

## Phase 4 - Admin Dashboard

- Add report moderation.
- Add project management.
- Add basic analytics.

## Out of Scope for MVP

- AI verification
- Flood prediction
- Satellite imagery comparison
- QR code verification
- Advanced analytics
- Push notifications
